# iot40-monitoring-system
oT 4.0 Monitoring System es una plataforma de monitoreo remoto basada en ESP32 + MQTT + Node.js + MongoDB + React, diseñada para recibir datos de sensores por internet, validarlos, almacenarlos y mostrarlos en una interfaz web en tiempo real mediante WebSocket. El sistema está organizado en módulos (backend, gateway, frontend)
